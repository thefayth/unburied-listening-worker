# Brand Style Notes

## Public Identity

Project name: Unburied Listening Worker

Public role: protected listening infrastructure for the Big Stick / Unburied /
Fightback ecosystem.

## Personality

- Protective
- Calm
- Serious
- Human
- Evidence-aware
- Local-first
- Boundary-conscious

## Color Direction

- Charcoal: `#171411`
- Warm umber: `#2b241f`
- Amber proof line: `#e0b15a`
- Protected green: `#76b7a1`
- Review rose: `#d684b8`
- Paper light: `#f7efe3`

## Visual Motifs

- Listening waves moving into protected review.
- A boundary line between raw intake and public-safe context.
- Mirror-safe export as a small approved summary, not a data spill.
- Warm control-room mood rather than generic SaaS.

## Avoid

- Surveillance-wall imagery.
- Police-state, panic, or extraction visuals.
- Legal documents, medical records, family photos, screenshots, case files, or
  readable private UI.
- "Product available now" visuals.
- Generic blue-purple gradient SaaS graphics.
