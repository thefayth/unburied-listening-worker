# How To Engage

## Appropriate

- Ask for private background context through Faith's official channels.
- Ask whether Faith is open to a private demo or partnership review.
- Reference the project only as a protected private-system component.
- Respect that public documentation does not grant usage rights.

## Not Appropriate

- Asking for source code, private prompts, raw data, transcripts, credentials,
  or deployment details.
- Treating this public surface as a product download.
- Scraping, cloning, packaging, training on, or commercializing the material.
- Probing private endpoints or systems.

## Contact

Use Faith's official public contact path. No private contact information is
included in this surface.
