# Project Brief: Unburied Listening Worker

## One-Sentence Description

Unburied Listening Worker is a protected local companion service for controlled
media, transcript, and social listening review inside the private Big
Stick/Unburied system.

## What It Is

Inspected files describe a private Node.js and Express worker named
`bigstick-social-service`. It supports authenticated social/listening intake,
uploaded subtitle/audio/video handling, transcript storage, local
rules-oriented analysis, search/export paths, and mirror-safe sync back into
the parent Big Stick/Unburied app.

## What It Is Not

- It is not a public app.
- It is not a SaaS offer.
- It is not a public transcript database.
- It is not an open-source release.
- It is not a download, checkout, or public API.

## Who It May Eventually Serve

Unknown from inspected files as a public offering. The private system appears
designed for Faith's Big Stick/Unburied review workflows and could support
trusted private review or partnership conversations if Faith chooses.

## Why It Matters

The worker separates raw intake from mirror-safe review data. That boundary
matters when transcripts, media, social posts, and evidence-like materials must
be reviewed without casually exposing private records.

## Current Public Status

Public-safe documentation is staged locally for Faith review. No public repo was
created and no WordPress page was published.

## Private Work That Remains Protected
