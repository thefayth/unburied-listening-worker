# FAQ

## Is this an open-source project?

No. This is a protected public project surface. No public license is granted.

## Can I use the worker?

No public access is offered through this surface.

## Is this a product for sale?

No. This surface does not sell anything and does not provide checkout, pricing,
downloads, or access.

## Why make a public surface at all?

To show that the project exists, has a serious privacy boundary, and can be
discussed safely without exposing private data or operational IP.

## What should stay private?

Source code, private workflows, transcripts, uploads, credentials, runtime
storage, deployment details, legal/admin/family records, and all sensitive
receipts.
