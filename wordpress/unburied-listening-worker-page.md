# Unburied Listening Worker

Protected listening infrastructure for private review and mirror-safe public
boundaries.

## What This Is

Unburied Listening Worker is a private companion service inside the Big
Stick/Unburied system. It supports controlled media, transcript, and social
listening workflows while keeping raw intake and review artifacts protected.

## Why It Matters

Listening work can contain sensitive context. This project is built around a
clear boundary: raw materials stay private, review happens in protected space,
and only mirror-safe metadata or summaries move outward after review.

## What Is Public

- High-level project story.
- Public/private boundary.
- Non-sensitive roadmap.
- Abstract visuals and image briefs.

## What Remains Private

Source code, transcripts, uploads, analysis artifacts, credentials, deployment
details, private receipts, and legal/admin/family/evidence-related records.

## Current Status

This is a protected project surface prepared for Faith review. It is not a
public app, product download, or open-source release.
