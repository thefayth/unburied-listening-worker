# Assets

This folder contains the public-safe visual package for the Unburied Listening
Worker GitHub surface and held WordPress draft.

Do not place private screenshots, transcripts, logs, recordings, credentials,
case records, addresses, names, IDs, or dashboard captures here.

Current assets:

- `unburied-listening-worker-system-diagram.svg`: abstract placeholder diagram
  with no private data.
- `brand/`: approved existing Big Stick, Unburied, and Fightback brand visuals.
- `hero/hero-image.png`: README and WordPress hero candidate.
- `banners/github-banner.png`: README/GitHub banner.
- `social/social-card.png`: social preview / Open Graph candidate.
- `icons/project-icon.png`: project icon / ecosystem marker.
- `diagrams/`: branded workflow and public/private boundary diagrams.
- `gallery/`: selected existing ecosystem visuals.

Prompt briefs live in `assets/image-prompts/`.
