# System Diagram Prompt

Filename suggestion: `unburied-listening-worker-system-diagram.png`

Purpose: Explain the public/private boundary without showing private systems.

Placement: Project brief, WordPress visual section.

Aspect ratio: 16:9 or 1200x630.

Style direction: Clean labeled diagram.

Prompt: Design a simple public-safe system diagram with three boxes: Private
Intake, Protected Review, Mirror-Safe Export. Use abstract icons for listening,
review, and approved metadata. Show that raw materials remain private and only
approved summaries move outward. Use warm dark neutrals with amber, green, and
rose accents.

Negative prompt / avoid: No endpoint names, no code, no server paths, no
credentials, no screenshots, no real documents, no personal data.

Alt text: Public-safe diagram showing private intake, protected review, and
mirror-safe export.

Privacy notes: Safe to generate now.
